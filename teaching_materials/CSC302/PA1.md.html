However, it involves the notion of recursion, which we will discuss later in the semester. Another way to solve the problem is by bitmasking, which does not involves recursion. It is sufficient to use this approach in the current assignment. If you feel comfortable and confident using recursion, you can also use backtracking. You can find a discussion of both algorithms in this [article](https://afteracademy.com/blog/print-all-subsets-of-a-given-set). Pseudo Codes are provided in the article. Note that pseudo codes are just plain language explaining algorithm design ideas. You still need to implement it in your chosen programming language (Java, C/C++, ...). Here is the pseudo code for the bitmasking algorithm, in case for any reason you can not open the above link. ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~java Void allSubset(int S[], int n) { int subset_size = pow(2, n) int index, i //Run from index 000..0 to 111..1 for(index from 0 to subset_size) { int subset[n] for(i from 0 to n) { // Check if jth bit in the index is set // If set then add jth element to the subset if(index & (1 << i)) subset[i] = S[i] } print(subset) } } ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ **Useful fact**. The size of the powerset of S of size $n$ is $2^n$. **Bitmasking**. You can find a short tutorial on bitmasking trick [here](https://discuss.codechef.com/t/a-small-tutorial-on-bitmasking/11811).
